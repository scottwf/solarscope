Get daily totals:
http://your-pi.local:5000/summary/daily?date=2025-04-10

Download usage as CSV:
http://your-pi.local:5000/power-usage?start=2025-04-10&end=2025-04-10&csv=true

To download generation as CSV:
http://your-pi.local:5000/solar-generation?start=2025-04-10&end=2025-04-10&csv=true

Will return:
timestamp,energy
2025-04-10 06:00:00,45.0
2025-04-10 06:15:00,72.3
